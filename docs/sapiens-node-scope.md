# 《人类简史》节点知识范围与掌握标准

> 本文档定义每个节点在 Knowledge World 中的知识边界、用户的掌握标准和常见误解，供 AI 诊断引擎和内容设计参考。
>
> **分层说明**：每个节点的 Scope 已拆分为 What / How / Why / System 四层，每层有明确的"包含"和"不包含"边界。What 层不设掌握判断，读完三张卷轴即完成。How / Why / System 三层各对应一条掌握标准，作为出题和判定依据。原问回响综合全部 3 条标准进行终判。

---

## 节点一：认知革命

**关卡 NPC：** 讲故事的人

【总体知识范围 Overview】
智人与其他动物的差异不在于个体体能或智力，而在于协作规模。关键是"虚构故事的能力"——能用语言描述不存在的事物，让陌生人相信同一套秩序，从而突破邓巴数字（约 150 人）。

---

**【What 层 Scope · 知道它是什么】**
- **包含**：虚构故事的定义；认知革命发生的时间（约七万年前）；智人语言和动物语言的区别；虚构和谎言的区别
- **不包含**：虚构如何促进协作的详细机制（How 层）；为什么虚构比个体能力重要（Why 层）；现代社会的例子（System 层）

**【How 层 Scope · 理解它如何运作】**
- **包含**：虚构故事促进协作的链条（共同相信 → 信任基础 → 大规模协作）；邓巴数字及其突破机制；部落神话 → 宗教 → 公司的机制演化
- **不包含**：虚构故事的定义（What 层）；为什么虚构比个体能力更重要（Why 层）；个人生活中的例子（System 层）

**【Why 层 Scope · 追问为什么成立】**
- **包含**：为什么个体能力不是智人胜出的关键；尼安德特人对比分析；反事实分析（如果没有虚构能力会怎样）；聪明是协作的结果而非原因
- **不包含**：虚构的具体定义（What 层）；协作的具体机制（How 层）；现代场景的连接（System 层）

**【System 层 Scope · 迁移到更大系统】**
- **包含**：虚构能力在现代世界中的体现（金钱、国家、公司、品牌）；认知革命 → 现代生活的连接；虚构如何支撑全球化协作
- **不包含**：虚构的定义（What 层）；虚构如何运作的机制（How 层）；因果分析（Why 层）

---

【掌握标准 Mastery Criteria】
用户如果能做到以下 3 点，就算掌握：
1. 能解释"为什么个体能力不是关键"（因为尼安德特人个体更强，但智人靠大规模协作胜出）
2. 能说明"虚构如何促进协作"（共同相信一个故事 → 陌生人互信 → 大规模协作）
3. 能用一个现实例子解释（如金钱：所有人都相信它值钱，所以它能流通）

【分层掌握标准 Depth Criteria】

| 深度 | 对应标准 | LLM 判定指引 |
|------|---------|-------------|
| **What** | 不设掌握判断 | 读完三张卷轴 + 选择关键卷轴 = `completed` |
| **How** | 标准②：说明"虚构如何促进协作" | 用户是否解释了从"共同相信一个故事"到"陌生人互信"再到"大规模协作"的链条 |
| **Why** | 标准①：解释"为什么个体能力不是关键" | 用户是否提到了尼安德特人对比，或说明了"协作规模而非个体能力"才是关键 |
| **System** | 标准③：用一个现实例子解释 | 用户是否用金钱/国家/公司等现实例子说明了虚构的运作方式 |
| **原问回响** | 综合全部 3 条 | 检查回答是否覆盖全部三层的 Scope 核心点 |

【常见误解 Misconceptions】
- 认为人类更聪明所以胜出（错在把"聪明"当原因，聪明是协作的结果）
- 认为是工具或火带来文明（工具在认知革命前就存在了）
- 认为是语言本身（动物也有语言，区别在于语言的种类——能否描述不存在的东西）
- 认为虚构＝谎言（虚构是共同想象的框架，不是欺骗）

## 节点二：农业革命

**关卡 NPC：** 老农夫

【总体知识范围 Overview】
农业革命不是人类的"进步"，而是一条不归路。对物种有利（人口爆炸），对个体不利（劳动量↑、营养↓、健康↓）。驯化是双向的：小麦驯化了智人。定居生活 → 财产积累 → 社会分层。

---

**【What 层 Scope · 知道它是什么】**
- **包含**：农业革命的定义和时间（约一万年前）；从采集狩猎到定居种植的转变；主要驯化的作物和动物
- **不包含**：农业革命为什么是"骗局"（Why 层）；双向驯化的详细机制（How 层）；与现代生活的类比（System 层）

**【How 层 Scope · 理解它如何运作】**
- **包含**：人驯化小麦 vs 小麦驯化人的机制；农业如何改变人类的饮食、骨骼健康和社会结构；人口增长 → 无法回头的正反馈循环
- **不包含**：农业的定义和时间线（What 层）；为什么是个体陷阱（Why 层）；现代场景类比（System 层）

**【Why 层 Scope · 追问为什么成立】**
- **包含**：为什么农业对物种有利但对个体有害；为什么人类无法回到采集生活；为什么说农业是"史上最大的骗局"
- **不包含**：农业的具体定义（What 层）；双向驯化的机制（How 层）；现代人的类比（System 层）

**【System 层 Scope · 迁移到更大系统】**
- **包含**：农业陷阱的现代版本（上班、房贷、稳定收入陷阱）；农业革命与后续文明（城市、文字、阶级）的连接；采集社会和现代社会的对比
- **不包含**：农业革命的定义（What 层）；机制解释（How 层）；因果分析（Why 层）

---

【掌握标准 Mastery Criteria】
用户如果能做到以下 3 点，就算掌握：
1. 能解释"农业是骗局"的含义（个体生活品质下降，但回不去了）
2. 能说明"人驯化小麦 vs 小麦驯化人"（小麦让人类困在田里，改变了人类的生活方式）
3. 能用一个现代类比说明（如上班＝现代版农业陷阱，用自由换稳定收入）

【分层掌握标准 Depth Criteria】

| 深度 | 对应标准 | LLM 判定指引 |
|------|---------|-------------|
| **What** | 不设掌握判断 | 读完三张卷轴 + 选择关键卷轴 = `completed` |
| **How** | 标准②：说明"人驯化小麦 vs 小麦驯化人" | 用户是否解释了驯化的双向性——人类以为在控制作物，实际被作物改变了生活方式 |
| **Why** | 标准①：解释"农业是骗局"的含义 | 用户是否说明了"对物种有利 ≠ 对个体有利"，以及为什么回不去 |
| **System** | 标准③：用一个现代类比说明 | 用户是否将农业陷阱迁移到现代场景（如上班/房贷/稳定收入陷阱） |
| **原问回响** | 综合全部 3 条 | 检查回答是否覆盖全部三层的 Scope 核心点 |

【常见误解 Misconceptions】
- 认为农业革命是人类主动的聪明选择（实际是被人口压力和机会推动的被动选择）
- 认为农业让生活更好（对个体而言是劳动时间增加、营养多样性下降）
- 认为采集生活是"原始的"（采集者工作时间更短、饮食更多样）
- 混淆"对物种好"和"对个体好"

## 节点三：货币的诞生

**关卡 NPC：** 商人

【总体知识范围 Overview】
货币的本质不是实物，而是共同想象的信任系统。形式不重要（贝壳、金币、纸币、数字支付），信任是核心。货币让完全陌生的两个人可以瞬间合作，信任网络覆盖全球。

---

**【What 层 Scope · 知道它是什么】**
- **包含**：货币的定义；以物易物的局限；货币作为交换媒介、价值尺度和贮藏手段；货币形式的演化（贝壳→金币→纸币→数字）
- **不包含**：货币为什么能被信任的深层机制（How 层）；为什么一张纸能买到食物（Why 层）；货币和宗教的对比（System 层）

**【How 层 Scope · 理解它如何运作】**
- **包含**：货币如何解决"需求双巧合"问题；货币作为通用媒介的运作机制；信任如何从熟人扩展到陌生人；货币的流通和信用体系
- **不包含**：货币的定义和历史（What 层）；信任的本质分析（Why 层）；与古代信仰的跨领域对比（System 层）

**【Why 层 Scope · 追问为什么成立】**
- **包含**：为什么一张纸/一枚金币能买到真实食物；"所有人相信它值钱"为什么足以支撑整个货币体系；货币为什么比以物易物更优越；信任的经济学本质
- **不包含**：货币的定义（What 层）；货币的具体运作机制（How 层）；全球化的连接（System 层）

**【System 层 Scope · 迁移到更大系统】**
- **包含**：货币和古代信仰/宗教的共同点（都是共同想象的信任网络）；从贝壳到比特币——信任形式的演变；货币的全球化如何连接不同文明
- **不包含**：货币的定义（What 层）；需求双巧合等机制（How 层）；信任的本质因果分析（Why 层）

---

【掌握标准 Mastery Criteria】
用户如果能做到以下 3 点，就算掌握：
1. 能解释"为什么一张纸能买到食物"（因为所有人都相信它代表价值）
2. 能说明货币和古代信仰的共同点（都是共同想象，区别在于货币不信也能用）
3. 能解释为什么货币比以物易物更有效（货币解决了"需求双巧合"问题）

【分层掌握标准 Depth Criteria】

| 深度 | 对应标准 | LLM 判定指引 |
|------|---------|-------------|
| **What** | 不设掌握判断 | 读完三张卷轴 + 选择关键卷轴 = `completed` |
| **How** | 标准③：解释货币比以物易物更有效 | 用户是否解释了"需求双巧合"问题，以及货币作为通用媒介如何解决它 |
| **Why** | 标准①：解释"为什么一张纸能买到食物" | 用户是否说到"所有人相信它值钱"而非货币本身有价值——信任的本质 |
| **System** | 标准②：说明货币和古代信仰的共同点 | 用户是否跨领域连接了货币和宗教/神话——两者都是共同想象的信任网络 |
| **原问回响** | 综合全部 3 条 | 检查回答是否覆盖全部三层的 Scope 核心点 |

【常见误解 Misconceptions】
- 认为货币有"内在价值"（黄金、纸币本身并没有使用价值）
- 认为货币是政府发明的（货币是自发形成的信任网络，政府后来才背书）
- 认为电子支付和实物货币本质不同（形式变了，信任机制不变）
- 认为只有现代社会才有货币（贝壳、石币等早期货币存在于几乎所有文明）

## 节点四：想象的秩序

**关卡 NPC：** 石碑书记

【总体知识范围 Overview】
想象的秩序：一群人共同相信一个不存在的东西，它因此变成"社会层面的真实"。包括神、国家、法律、公司、人权、版权、品牌。现代生活的底层代码就是这些想象出来的秩序。

---

**【What 层 Scope · 知道它是什么】**
- **包含**：想象的秩序的定义；想象的秩序 vs 物理真实的区别；想象的秩序的例子（国家、法律、公司、人权）
- **不包含**：想象的秩序如何运作的机制（How 层）；为什么它与谎言不同（Why 层）；它在现代生活中的维持系统（System 层）

**【How 层 Scope · 理解它如何运作】**
- **包含**：想象的秩序"如何"产生真实力量；公司作为法律虚构人的运作机制；品牌价值的形成过程；从石头记帐到现代会计的演化
- **不包含**：想象的秩序的定义（What 层）；与谎言的本质区别（Why 层）；维持机制的系统分析（System 层）

**【Why 层 Scope · 追问为什么成立】**
- **包含**：为什么想象的秩序和"谎言"不同；为什么大家都知道它是虚构的但仍然选择相信；集体认同的本质；想象的秩序为什么比刀剑更有力量
- **不包含**：想象的秩序的定义（What 层）；它的具体运作机制（How 层）；现代社会如何维持（System 层）

**【System 层 Scope · 迁移到更大系统】**
- **包含**：想象的秩序如何通过教育、仪式、执法、重复叙事来维持；数字时代产生了哪些新的想象的秩序（品牌、平台规则、社群认同）；现代社会和古代信神的结构同源性
- **不包含**：想象的秩序的定义（What 层）；运作机制（How 层）；因果本质分析（Why 层）

---

【掌握标准 Mastery Criteria】
用户如果能做到以下 3 点，就算掌握：
1. 能举例说明一个"想象的秩序"并解释它如何运作（如公司：一个法律虚构的人）
2. 能解释为什么"想象的秩序"和"谎言"不同（所有人都知道它是虚构的，但都选择相信）
3. 能说明想象的秩序如何维持（通过教育、仪式、执法、重复叙事）

【分层掌握标准 Depth Criteria】

| 深度 | 对应标准 | LLM 判定指引 |
|------|---------|-------------|
| **What** | 不设掌握判断 | 读完三张卷轴 + 选择关键卷轴 = `completed` |
| **How** | 标准①：举例说明想象的秩序并解释运作方式 | 用户是否举了一个具体例子（公司/国家/品牌）并解释了它"如何"发挥真实力量 |
| **Why** | 标准②：解释想象的秩序和谎言的区别 | 用户是否说清了"集体认同"这个关键——不因为它是假的就不起作用 |
| **System** | 标准③：说明想象的秩序如何维持 | 用户是否连接了教育、仪式、法律等更大系统来解释维持机制 |
| **原问回响** | 综合全部 3 条 | 检查回答是否覆盖全部三层的 Scope 核心点 |

【常见误解 Misconceptions】
- 认为"想象的"= "不存在的"（在物理层面不存在，但在社会层面真实存在）
- 认为现代社会已经摆脱了"迷信"（现代人每天遵守的秩序和古人信神结构相同）
- 认为想象的秩序是被强加的（更多是集体默认，而非少数人控制）
- 混淆"想象的秩序"和"意识形态"（想象的秩序是更底层的框架，意识形态是上层表达）

## 节点五：资本主义与法律虚构

**关卡 NPC：** 银行家

【总体知识范围 Overview】
法律虚构（Legal Fiction）：没有身体但有权利和责任的"人造人"。公司是不死的、可以无限扩张的法律虚构实体。股票、债券、商标、专利都建立在法律虚构之上。资本主义的核心不是赚钱，而是"必须增长"。

---

**【What 层 Scope · 知道它是什么】**
- **包含**：法律虚构的定义；公司作为法律虚构人的概念；资本主义的基本定义；股票、债券、商标、专利是什么
- **不包含**：公司如何运作的机制（How 层）；为什么纸上规则有约束力（Why 层）；与古代神话的对比（System 层）

**【How 层 Scope · 理解它如何运作】**
- **包含**：公司如何拥有财产、签合同、起诉（法律运作机制）；公司不死亡、无限扩张的法律原理；股票市场的运作基础；资本主义"必须增长"的机制
- **不包含**：法律虚构的定义（What 层）；纸上规则为什么有约束力（Why 层）；与古代神话的类比（System 层）

**【Why 层 Scope · 追问为什么成立】**
- **包含**：为什么纸上规则有约束力（因为集体承认）；为什么公司这个"人造人"能拥有真实力量；为什么资本主义要求"必须增长"
- **不包含**：法律虚构的定义（What 层）；法律运作机制（How 层）；跨领域连接（System 层）

**【System 层 Scope · 迁移到更大系统】**
- **包含**：法律虚构和古代神话/宗教的相似性（都是"信则有"的共同规则）；你的工资、房贷、股票如何建立在法律虚构之上；法律虚构如何塑造了现代社会结构
- **不包含**：法律虚构的定义（What 层）；运作机制（How 层）；因果分析（Why 层）

---

【掌握标准 Mastery Criteria】
用户如果能做到以下 3 点，就算掌握：
1. 能解释"公司是一个人造人"的含义（没有身体但有财产权，不死亡可无限扩张）
2. 能说明法律虚构和古代神话的相似性（都是共同相信的规则，都有真实力量）
3. 能用一个日常例子说明（如你的工资、房贷、股票都建立在法律虚构之上）

【分层掌握标准 Depth Criteria】

| 深度 | 对应标准 | LLM 判定指引 |
|------|---------|-------------|
| **What** | 不设掌握判断 | 读完三张卷轴 + 选择关键卷轴 = `completed` |
| **How** | 标准①：解释"公司是一个人造人"的含义 | 用户是否说清了法律虚构的机制——一个没有身体但有权利的人造实体如何运作 |
| **Why** | 标准③：用日常例子说明法律虚构的力量 | 用户是否追问了"为什么"这些纸上规则有约束力——因为集体承认 |
| **System** | 标准②：说明法律虚构和古代神话的相似性 | 用户是否跨领域连接了法律虚构和神话——两者都是"信则有"的共同规则 |
| **原问回响** | 综合全部 3 条 | 检查回答是否覆盖全部三层的 Scope 核心点 |

【常见误解 Misconceptions】
- 认为资本主义＝市场经济（市场经济自古存在，资本主义是近代产物）
- 认为法律虚构是理所当然的（古代人会觉得"一个没有身体的人"很荒谬）
- 混淆"公司"和"工厂"（公司是法律概念，工厂是物理空间）
- 认为资本主义只关乎经济（它改变了社会结构、时间观念和生活方式）

## 节点六：帝国的崛起

**关卡 NPC：** 帝王使者

【总体知识范围 Overview】
帝国通过武力扩张，但靠文化维持统治。核心悖论：军事征服者，最终被被征服者的文化同化。文字、法律、宗教、哲学是帝国真正的统治工具。帝国留下的文化遗产至今仍在影响我们。

---

**【What 层 Scope · 知道它是什么】**
- **包含**：帝国的定义；帝国统治的核心特征（扩张、多元文化、统一制度）；主要帝国的例子（罗马、蒙古、大英）
- **不包含**：征服者被同化的机制（How 层）；为什么武力赢不了文化（Why 层）；今天的帝国遗产（System 层）

**【How 层 Scope · 理解它如何运作】**
- **包含**：帝国如何用武力征服 + 文化统治（双重管理机制）；征服者被同化的具体过程（罗马学希腊、蒙古学中国）；文化渗透比军事占领更持久的原因
- **不包含**：帝国的定义（What 层）；核心悖论的原因分析（Why 层）；现代连接（System 层）

**【Why 层 Scope · 追问为什么成立】**
- **包含**：为什么武力赢土地但文化赢人心；为什么征服者需要被征服者的制度来治理；为什么使用对方的语言/宗教/法律会导致被同化；征服与统治的根本区别
- **不包含**：帝国的定义（What 层）；同化的具体过程（How 层）；今天的帝国遗产（System 层）

**【System 层 Scope · 迁移到更大系统】**
- **包含**：帝国留在今天的遗产（语言、法律体系、宗教传播、世界观的跨文化融合）；现代全球化中帝国逻辑的延续；文化影响超越政治边界
- **不包含**：帝国的定义（What 层）；同化机制（How 层）；因果分析（Why 层）

---

【掌握标准 Mastery Criteria】
用户如果能做到以下 3 点，就算掌握：
1. 能解释"为什么征服者被反噬"（武力赢土地，文化赢人心；治理需要被征服者的成熟体系）
2. 能举例说明（罗马学希腊、蒙古学中国）
3. 能说明今天的哪些东西是帝国遗产（语言、法律体系、文字、宗教传播）

【常见误解 Misconceptions】

## 节点七：科学革命

**关卡 NPC：** 观星者

【总体知识范围 Overview】
科学革命的核心不是获得新知识，而是承认无知。之前所有文明都认为真理已写好。"我不知道"从羞耻变成美德，从终点变为起点。观察 + 实验 + 数学 → 自我修正的系统 → 无上限的知识增长。

---

**【What 层 Scope · 知道它是什么】**
- **包含**：科学革命的定义和时间（约公元1500年起）；"承认无知"的含义；科学方法的基本步骤；科学革命前后的态度对比
- **不包含**：科学方法的具体运作（How 层）；为什么承认无知是力量（Why 层）；生活中如何使用科学方法（System 层）

**【How 层 Scope · 理解它如何运作】**
- **包含**：科学方法的具体运作过程（观察→假设→实验→验证→修正）；从"答案在书里"到"我自己验证"的机制转变；科学共同体如何自我纠错（同行评审、可重复性）
- **不包含**：科学革命的定义（What 层）；为什么无知带来力量（Why 层）；个人日常应用（System 层）

**【Why 层 Scope · 追问为什么成立】**
- **包含**：为什么承认无知反而带来力量；为什么之前的文明都认为真理已写好；正反馈循环（不知道→探索→新发现→更多不知道）的因果逻辑；科学和宗教关系的复杂性
- **不包含**：科学革命的定义（What 层）；科学方法的具体步骤（How 层）；个人生活应用（System 层）

**【System 层 Scope · 迁移到更大系统】**
- **包含**：你在日常生活中如何使用科学方法（查资料验证、看评测、不轻信单一来源）；科学革命如何与资本主义和帝国相互强化；现代人每天在手机上实践科学革命
- **不包含**：科学革命的定义（What 层）；科学方法的机制（How 层）；因果分析（Why 层）

---

【掌握标准 Mastery Criteria】
用户如果能做到以下 3 点，就算掌握：
1. 能解释"承认无知为什么是力量"（承认不知道→去探索→发现新知识→永不停歇）
2. 能说明科学革命前和科学革命后的核心区别（从"一切答案在书里"到"我来验证"）
3. 能说明自己在生活中如何使用科学方法（查资料验证、看评测买东西、不相信单一来源）

【分层掌握标准 Depth Criteria】

| 深度 | 对应标准 | LLM 判定指引 |
|------|---------|-------------|
| **What** | 不设掌握判断 | 读完三张卷轴 + 选择关键卷轴 = `completed` |
| **How** | 标准②：说明科学革命前后的核心区别 | 用户是否解释了从"答案在书里"到"我自己验证"这个机制转变 |
| **Why** | 标准①：解释"承认无知为什么是力量" | 用户是否说清了"不知道→去探索→新发现"这个正反馈循环的原因 |
| **System** | 标准③：说明自己生活中如何使用科学方法 | 用户是否迁移到了个人日常——查资料、验证信息、不轻信单一来源 |
| **原问回响** | 综合全部 3 条 | 检查回答是否覆盖全部三层的 Scope 核心点 |

【常见误解 Misconceptions】
- 认为科学革命是一堆发明（发明是结果，不是原因；态度转变才是革命）
- 认为科学和宗教是对立的（历史上科学和宗教的关系远比"对立"复杂）
- 认为科学是西方独有的（数学、天文学等基础来自阿拉伯、印度和中国）
- 认为科学能回答一切问题（科学的边界是它只回答可验证的问题）
- 认为科学＝技术（科学是知识体系，技术是应用；现代技术是科学与工程的结合）
