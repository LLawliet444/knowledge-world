# 像素风素材生成规范

> 本规范是 Knowledge World 视觉资源的唯一来源准则。M1 阶段所有生成、外采、手绘的像素素材都必须遵守本文件。

文档创建日期：2026-06-17
所属里程碑：M1 视觉基线（06-18 ~ 06-24）
责任人：视觉负责人 + AI 生成 Pipeline 负责人

---

## 1. 设计目标

像素风不是装饰，而是产品语义的一部分。Knowledge World 的像素风必须同时满足三件事：

1. 一眼能看出是"探索类游戏"，唤起《星露谷物语》《Octopath Traveler》式的探索欲
2. 调性温暖、叙事化，不能像复古横版游戏那样冷硬
3. 在 1366×768 ~ 1920×1080 桌面端浏览器中保持清晰，不出现"AI 伪像素"颗粒感

---

## 2. 风格圣经（一切素材必须对齐）

风格圣经是项目的视觉基线，由 1 张参考底图 + 1 套调色板 + 1 段 base prompt 组成。任何素材在生成或外采前都必须先对齐这份圣经。

### 2.1 像素密度规范

| 资源类型 | 像素密度 | 最终输出尺寸 |
|---|---|---|
| 生物群系底图（4 张大场景） | 等效 256×144 像素栅格 | 1920×1080，最近邻放大 |
| 节点图标 | 32×32 像素 | 64×64 展示，最近邻放大 |
| 学者化身 | 16×16 ~ 24×24 像素 | 48×48 展示 |
| 徽章 / UI 图标 | 32×32 像素 | 64×64 展示 |
| 迷雾粒子贴图 | 16×16 像素 | 32×32 展示 |

整个项目只允许出现上述 5 种像素密度，禁止混用。

### 2.2 调色板规范

总调色板限制为 24 色，按四个生物群系划分主色：

| 层级 | 生物群系 | 主色 HEX | 用途 |
|---|---|---|---|
| What | 认知大草原 | `#F5B642`（暖黄）| 入口、基础概念节点 |
| How | 结构金绿林 | `#78D98B`（金绿）| 模块结构、路径节点 |
| Why | 因果石灰原 | `#B5A88A`（石灰）| 原理、挑战节点 |
| System | 系统紫蓝海 | `#8E6CFF`（紫蓝）| 跨域连接、迷雾覆盖 |

通用色（背景、阴影、UI）共 8 色，参考 `submission/knowledge-world-creative/knowledge-world-creative.html` 中的 CSS 变量。

> 所有素材最终必须量化到这 24 色。生成阶段允许偏色，但导出前一定要做一次调色板量化（见 §5.3）。

### 2.3 透视与描边

- 透视：统一 3/4 俯视角，类似《星露谷物语》。禁止正侧视、纯俯视。
- 描边：节点图标、化身、徽章使用硬描边（1 像素深色边）；生物群系底图无描边，靠色块和明暗划分。
- 光源：左上 45 度方向，所有阴影朝右下。

### 2.4 Base Prompt（所有 AI 生成的固定前缀）

```text
pixel art, 32x32 sprite, 3/4 top-down view, warm narrative game style,
soft pastel palette of yellow #F5B642, green #78D98B, beige #B5A88A, purple #8E6CFF,
hard 1px dark outline, light source from top-left,
Stardew Valley meets Octopath Traveler, no anti-aliasing, no gradient,
clean readable silhouette, single subject centered
```

任何生成都必须在此基础上拼接具体内容，不允许另起 prompt。

| 素材类型 | 数量 | 推荐工具 |

### 3.3 P2 不做（留到复赛）

### 3.4 工时分配建议（M1 06-18 ~ 06-24，共 6 天）

| 资源 | 工时占比 | 说明 |
|---|---|---|
| 风格圣经定稿 | 15% | 一次性投入，决定后续效率 |
| 4 张生物群系底图 | 25% | 用 SDXL 批量生成 + 筛选，2 人天 |
| 12 个节点图标（6 专属 + 6 通用） | 25% | 数量多 + 筛选成本高，2 人天 |

## 4. 生成与处理流水线

为保证风格一致性，所有 AI 生成素材必须走以下三层流水线，缺一不可。

### 4.1 第一层：风格圣经定稿（M1 第 1 天）

1. 视觉负责人用 Base Prompt + "认知大草原入口村，黄昏暖光，远处有像素小屋" 生成 8 张候选底图
2. 评审会从中选出 1 张作为风格基线参考图
3. 提取调色板，写入 `assets/style/palette.png`（24 色的色板小图）
4. Base Prompt 与调色板锁死，进入第二层

### 4.2 第二层：批量生成（M1 第 2–4 天）

- **每个素材至少出 8–12 张候选**，禁止"一次出图就用"。像素风对单像素错误极敏感
- 候选用 ComfyUI 或脚本批量生成，命名格式：`{类型}_{编号}_{seed}.png`
- 视觉负责人逐张审，挑出 1 张主图 + 1 张备选

### 4.3 第三层：后期处理（每张图必做）

为消除"AI 伪像素"，每张生成图导出前必须做这两步：

1. **强制降采样**：用 [Pixelator](https://www.pixelator.co/) 或 ImageMagick 命令降采样到目标像素密度
   ```bash
   convert input.png -filter point -resize 32x32 -filter point -resize 512x512 output.png
   ```
2. **调色板量化**：用 ImageMagick 或 Aseprite 强制量化到 §2.2 的 24 色调色板
   ```bash
   convert input.png -dither None -remap palette.png output.png
   ```

未做这两步的资源不允许进入仓库 `assets/` 目录。

## 5. 角色与动画特殊约定

学者化身是产品记忆点之一，标准更严格：

- 外采优先：在 itch.io 上有大量 5–20 美元的 16×16 / 24×24 学者风格角色精灵图，附带 4 方向 × 4 帧走路动画。优先购买使用
- 自绘兜底：如外采无合适角色，由视觉负责人在 Aseprite 手绘，1 帧站立 + 4 方向各 4 帧走路 = 17 帧
- 颜色替换：外采角色一律用调色板替换工具改成 §2.2 的主调色板，不允许出现额外颜色
- 严禁 AI 生成动画：AI 生成的逐帧动画在像素风下会出现明显抖动，本项目禁止使用

---

## 6. 迷雾系统视觉规范

迷雾是路演 demo 的视觉钩子，规范如下：

- 静态迷雾贴图：紫蓝色（`#8E6CFF`）+ 60% 透明度，16×16 平铺贴图，4 张轮换帧
- 消散动画：通过 Canvas/PixiJS 粒子系统实时合成，禁止预渲染 GIF
- 粒子参数：50–80 个粒子，半径 1–2 像素，初速度向外发散，1 秒淡出
- 颜色梯度：从 `#8E6CFF` 渐变到透明，中间过一次白色（0.3 秒）作为高光

所有像素素材统一保存在 `assets/` 下，结构如下：

├── biomes/                    # 4 张生物群系底图

│   ├── biome_how_forest.png
│   ├── biome_why_stoneland.png
│   └── biome_system_void.png

│   ├── node_cave_painting.png

├── fog/
│   ├── fog_tile_{1-4}.png
│   ├── fog_particle.png             # 迷雾消散粒子贴图

│   ├── halo_yellow.png              # 理解中

## 8. 验收清单（M1 出口标准）

### 8.1 P0 必须全部满足（决定 M1 能否出口）

- 风格圣经定稿（参考图 + 调色板 + Base Prompt）已入仓
- 4 张生物群系底图全部完成，且可拼接成完整 16:9 世界地图
- 12 个节点图标完成（其中至少 6 个为《人类简史》核心概念专属图标，避免地图机械感）
- 学者化身静帧 + 4 方向走路动画（17 帧）完成
- NPC 老学者头像 1 张完成，可在对话框中正常显示
- 4 张迷雾平铺贴图 + 1 张迷雾粒子贴图 + 1 张解锁粒子贴图完成
- 3 张状态光晕贴图（黄/绿/紫）完成
- "探险家绘制地图"等待动画 8 帧完成
- 5 个通用 UI 图标完成（外采或自绘均可）
- 所有素材通过 §4.3 的降采样与调色板量化处理
- 视觉负责人 + 产品负责人共同评审通过

### 8.2 P1 完成度记录（不阻塞出口，但建议在路演前补齐）

- NPC 对话框边框贴图
- 入口村引路 NPC
- 4 个区域征服徽章
- 1 张《人类简史》世界封面图

### 8.3 demo 第一镜头自检

M1 结束时，必须能用现有资源在 60 秒内拼出以下镜头：

> 学者站在迷雾笼罩的认知大草原入口村，向第一个节点行走，到达后弹出 NPC 对话框，回答后迷雾消散，邻接节点亮起。

如果该镜头无法拼出，即使 8.1 全部打钩也视为 M1 未出口，必须延期补齐。

## 9. 风险与降级

| 风险 | 概率 | 缓解 |
|---|---|---|
| AI 生成节点图标风格漂移 | 高 | 锁死 Base Prompt + 同 seed 批量出图 + 人工筛选 |
| 学者化身找不到合适外采资源 | 中 | 预算 1 天 Aseprite 手绘兜底 |
| 调色板量化后图标失真 | 中 | 在 palette 中预留 2 个高亮色专用于图标点缀 |
| Midjourney/SDXL API 不稳定 | 中 | 双工具并行，任一可单独完成生物群系底图 |
| 迷雾粒子动画在低端机掉帧 | 中 | 提供 CSS 关键帧降级版（透明度淡出 + 缩放） |
