# 对话框视觉哲学：羊皮纸低语

“羊皮纸低语”把界面视为知识世界里的一件器物，而不是覆盖在画面上的普通弹窗。它的形体来自旧地图、边角磨损的卷轴、发光的路径节点和导师留下的批注：所有信息都像从地图边缘浮现出来，轻声推动用户继续思考。文字只承担最少的锚定作用，真正的情绪由边框、暗角、发光节点和纸张肌理完成。

空间上，对话框必须有“落在地图上”的重量。边框不追求圆润的现代感，而采用像素化阶梯角、双层深棕描边、内侧暖金高光，让它像一块被精心裁切的交互地图碎片。头像区域不做真实人脸，而是做成导师印章或徽章式入口，既保留苏格拉底导师的存在感，也避免它变成普通聊天头像。

颜色遵循当前项目风格基线：羊皮纸米色作为主体，深棕作为结构，暖金作为引导，紫蓝作为未知与迷雾。紫蓝不应大面积压住阅读区，只在阴影、角标、节点微光处出现，像 System 层的远方雾气渗进当前问题。整个界面应看起来经过反复打磨，边缘、像素点、发光节奏都要有手工校准的痕迹。

节奏上，对话框分为三层：左侧导师印章、中央问题纸面、底部反馈与选择容器。三者不靠分割线硬切，而通过明暗层级和细小像素节点建立秩序。按钮和继续提示不应像现代 Web 按钮，而应像地图上的小路标、封蜡、金币或发光节点，让用户感觉自己不是在“点击 UI”，而是在推进探索。

最终成品必须像由顶级像素 UI 设计师花了很多小时逐点修出来：每一个角、每一条边、每一个高光都服务于“知识迷雾被提问照亮”的主题。它应该能独立作为一张游戏 UI 设定稿观看，也能拆成工程贴图直接进入 demo。
