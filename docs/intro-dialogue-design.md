# 节点入口引入学者-老学者对话设计方案

> 在每个节点过场动画结束后、What 翻卡之前，加入一段学者（玩家角色）与老学者之间的引导对话，让知识引入更自然、更有沉浸感。

文档创建日期：2026-06-18
状态：设计待实现

---

## 1. 当前问题

节点进入流程目前为：
```
点击节点 → 学者走到节点旁 → [首次] 播放过场动画（关卡NPC提出谜题）
                                              ↓
                                      弹对话框 → 直接展示What三张知识卡
```

当前流程中，过场动画结束后直接跳到翻卡，缺少学者（玩家角色）与老学者（导师）之间的互动环节来引入知识。玩家还没有机会消化谜题，就面对三张信息密集的卡片。

## 2. 修改后流程

```
点击节点 → 学者走到节点旁 → [首次] 播放过场动画（关卡NPC提出谜题）
                                              ↓
                                     学者-老学者引导对话（6句左右）
                                              ↓
                                      老学者展开三张知识卡（WhatCards）
```

关键变化：在过场和翻卡之间插入一段线性剧本对话，让学者代表玩家提问，老学者引导性地将话题自然过渡到知识卡。

## 3. 数据模型变更

### 3.1 MentorPrompts 扩展

在 `MentorPrompts` 中新增 `whatDialogue` 字段：

```typescript
export interface MentorPrompts {
  whatIntro: string;              // 已有：老学者简短开场白
  whatDialogue?: DialogueLine[];  // 新增：过场后引导对话
  how: string;
  why: string;
  system: string;
  finalReturn: string;
}

export interface DialogueLine {
  speaker: 'scholar' | 'mentor';
  text: string;
}
```

**向后兼容：** 如果节点没有提供 `whatDialogue`，跳过对话阶段，直接进入 WhatCards（现有行为不变）。

### 3.2 DialogPhase 扩展

```typescript
export type DialogPhase = 
  | "closed"
  | "intro_dialogue"   // 新增：学者-老学者对话阶段
  | "loading"
  | "reading"
  | "what_confirm"
  | "question"
  | "feedback"
  | "final";
```

### 3.3 DialogStore 新增 action

```typescript
interface DialogActions {
  // ... 现有 actions
  
  startIntroDialogue: () => void;  // 设置 phase 为 "intro_dialogue"
}
```

## 4. 对话界面示意

```
┌──────────────────────────────────────────────────────┐
│  📜 认知革命 · What · 认知层                    ✕  │
│                                           [跳过对话]  │
├──────────────────┬───────────────────────────────────┤
│                   │                                   │
│  当前说话者头像     │  [老学者]                           │
│  (高亮 96×96)     │  区别不在有没有语言，                    │
│                   │  而在语言的种类…                       │
│  ─────────        │                                   │
│                   │          [▾ 点击继续]                   │
│  另一方头像(暗)    │                                   │
│                   │                                   │
│                   │  最后一句后显示按钮：                   │
│                   │  [老学者展开三张知识卡 →]                │
└──────────────────┴───────────────────────────────────┘
```

### 4.1 界面变化

当前 DialogBox 左侧会动态显示当前说话者头像：
- 左上角 "关闭" 按钮同行右侧增加 **「跳过对话」** 链接（小字，不影响关闭按钮）
- 左侧列：当前对话者头像高亮，非对话者头像缩小/变暗
- 右侧内容区：说话者名字 + 逐字打字文本
- 底部：点击继续箭头提示

### 4.2 交互规则

| 规则 | 说明 |
|------|------|
| 逐字打字 | 45ms/字，点击未完成文本 → 立即显示全文 |
| 完整后点击 | 跳到下一句 |
| 头像切换 | 当前说话者头像高亮，另一方头像变暗至 40% |
| 最后一句 | 点击后显示 **「老学者展开三张知识卡 →」** 按钮 |
| 跳过对话 | 点击后直接进入 WhatCards，等同于对话已完成 |
| 对话完成后 | phase 设为 loading → setReading → 进入 WhatCards |
| 节点回访 | 已通关节点不再触发对话，直接进入 WhatCards |

## 5. 7 个节点对话剧本

### ① 认知革命 · 篝火洞穴

**谜题：** 是什么让智人和其他动物有了根本的不同？

| # | 说话者 | 台词 |
|---|--------|------|
| 1 | 学者 | "老爷爷问'智人到底哪里不一样'……我第一反应是'更聪明'，但这个答案好像太敷衍了。" |
| 2 | 老学者 | "你感觉没错。智人的大脑不比尼安德特人大，真正的差别不是硬件，而是我们用的'软件'——语言。" |
| 3 | 学者 | "动物不也有语言？蜜蜂跳舞、猴子报警，这不也是在交流吗？" |
| 4 | 老学者 | "区别在种类。动物的语言只能说真实的东西——'那边有吃的'。而智人的语言能说'不存在'的东西。" |
| 5 | 老学者 | "神、传说、规则、国家——这些看不见的东西，智人能靠语言描述出来，而且能让一群人共同相信。这就是我们最厉害的地方。" |

---

### ② 农业革命 · 金色麦田

**谜题：** 为什么定居务农的我们，反而比流浪的猎人还要累？

| # | 说话者 | 台词 |
|---|--------|------|
| 1 | 学者 | "农夫说日子比流浪时还苦……农业不是人类最伟大的进步之一吗？" |
| 2 | 老学者 | "对'人类'这个物种来说是进步——人口翻了百倍。但对'每一个具体的人'呢？你自己想一下：每天从早到晚种地，吃的东西就那几样，还动不动饿死。" |
| 3 | 学者 | "所以说，对个体来说是更辛苦了……但为什么大家还是要种地？" |
| 4 | 老学者 | "因为开弓没有回头箭。一旦开始种地、人口变多、变成村落，你就回不去采集生活了。农业就像一桩糟糕的交易——智人以为自己赢了，其实是亏了。" |
| 5 | 老学者 | "很多历史学家管这叫'史上最大的骗局'。到底是人驯化了小麦，还是小麦驯化了人，看完这三张卡你就有答案了。" |

---

### ③ 货币 · 热闹市集

**谜题：** 为什么一张不能果腹的纸，能买到最真实的食物？

| # | 说话者 | 台词 |
|---|--------|------|
| 1 | 学者 | "一张纸凭什么能买到包子？明明它自己就是一张纸。这个问题我一直觉得有点神奇。" |
| 2 | 老学者 | "神奇就对了。金钱的秘密就在于：**所有人都相信它值钱**。不是因为它本身有什么价值，而是因为每个人都这么信。" |
| 3 | 学者 | "那不就是'大家约定好'？和法律、规则差不多吗？" |
| 4 | 老学者 | "对，但比法律更厉害。法律只能管一个国家的范围。而金钱——你拿一张人民币，在美国、非洲、南极都能花。世界上没有第二种东西有这么大的信任网络。" |
| 5 | 老学者 | "金钱是人类最成功的'共同想象'，比任何宗教的信徒都多。它让完全陌生的人也能立刻合作。" |

---

### ④ 想象的秩序 · 远古神庙

**谜题：** 为什么一个看不见摸不着的神明，能让成千上万陌生人信任彼此？

| # | 说话者 | 台词 |
|---|--------|------|
| 1 | 学者 | "古代人信神我能理解，但咱们现代人应该不会这样了吧？" |
| 2 | 老学者 | "那我问你：法律不是写在纸上的吗？为什么它能有约束力？公司不是一个名字吗？为什么它能拥有财产、起诉别人？" |
| 3 | 学者 | "……嗯，因为大家都承认这些规则。" |
| 4 | 老学者 | "这不就回到和'信神'一样了吗？——所有人共同承认一个看不见的东西存在，它就真的存在了。国家、法律、公司、人权，本质上和神一样的。" |
| 5 | 老学者 | "区别只是：我们不再叫它'神'，叫它'制度'。但这些制度的力量，不比任何神明小。" |

---

### ⑤ 资本主义 · 石碑律法

**谜题：** 为什么写在纸上的规则，能驱使千万人为之日夜运转？

| # | 说话者 | 台词 |
|---|--------|------|
| 1 | 学者 | "规则写在纸上就有约束力，这我平时不觉得奇怪。但仔细一想……合同里写的字凭什么让我还钱呢？" |
| 2 | 老学者 | "因为整个社会都承认这个规则。但你有没有想过更古怪的？——公司有财产、能签合同、能起诉你，但公司本身没有身体，没有脑子。它是个'法律虚构的人'。" |
| 3 | 学者 | "法律虚构的人……就是说公司跟真人一样有权利，但它不是真人？" |
| 4 | 老学者 | "对。股票、商标、专利，甚至你今天用的APP背后，都是这种'人造人'在运作。现代世界的底层代码，就是这些法律虚构。" |
| 5 | 老学者 | "法律比刀剑更有力量——因为刀剑只能管住人，而法律能管住想象。" |

---

### ⑥ 帝国 · 帝国城门

**谜题：** 为什么军事上的征服者，最终总会被被征服者的文化反向吞噬？

| # | 说话者 | 台词 |
|---|--------|------|
| 1 | 学者 | "罗马征服了希腊，结果罗马人开始学希腊语。蒙古人征服了中国，反过来变成了中国人……征服者怎么被反杀了？" |
| 2 | 老学者 | "你说的这个现象本身就是答案——武力能占领土地，但征服不了人心。要治理一块地，你需要文字、法律、宗教、文化。" |
| 3 | 学者 | "但被征服的民族的文化，不就是'输家'的文化吗？为什么赢家反而要学？" |
| 4 | 老学者 | "因为你可能需要用对方的文字来收税，用对方的宗教来安抚人心，用对方的法律来断案。时间久了，征服者自己也被同化了。" |
| 5 | 老学者 | "雄狮走进笼子，以为自己征服了铁笼，其实是住进了别人建好的房子里。帝国最终的结局，往往是被自己征服的文明所改变。" |

---

### ⑦ 科学革命 · 星空观测台

**谜题：** 为什么承认无知，反而成了人类最强大的力量？

| # | 说话者 | 台词 |
|---|--------|------|
| 1 | 学者 | "承认无知反而成为力量……这很反直觉。从小到大我们都被教育要自信、要懂答案、不要说'我不知道'。" |
| 2 | 老学者 | "你说到了关键。科学革命之前，所有文明都认为真理早就写好了——在经书里、在古书里、在祖宗的智慧里。'我不知道'是丢脸的。" |
| 3 | 学者 | "科学的态度就是：古人说的不一定都对，我们要自己去验证？" |
| 4 | 老学者 | "对。而且更重要的：科学承认'我现在知道的，也可能错'。一旦你接受自己不知道，你就会去探索，而探索会带来新的发现。永远不会停止。" |
| 5 | 老学者 | "旧时代说：'一切答案都在书里。'科学说：'我不知道，所以我要去找。'——就是这个转变，让人类在过去两百年里，进步的速度超过了之前的所有时代。" |

## 7. 验收标准

- [ ] 首次进入节点时，过场动画结束后，学者-老学者对话自动开始
- [ ] 对话按顺序逐句播放，带逐字打字效果
- [ ] 未完成时点击→立即显示全文；已完成后点击→跳到下一句
- [ ] 当前说话者头像高亮，对方头像变暗
- [ ] 最后一句显示「老学者展开三张知识卡 →」按钮
- [ ] 点击「跳过对话」直接进入 WhatCards
- [ ] 对话结束后正常展示 WhatCards
- [ ] 节点已通关后回访不再触发对话
- [ ] 向下兼容：无 `whatDialogue` 的节点跳过对话阶段
