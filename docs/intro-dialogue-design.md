# 节点入口引入学者-老学者对话 + 三卷轴设计方案

> 在每个节点过场动画结束后、What 翻卡之前，加入一段学者（玩家角色）与老学者之间的引导对话，对话结束后老学者引出三张卷轴在对话框上方展开，用户逐一阅读，最后简短收尾对话后进入"选择关键卷轴"确认。

文档创建日期：2026-06-18
状态：设计待实现

---

## 1. 当前问题

节点进入流程目前为：

```
点击节点 → 学者走到节点旁 → [首次] 播放过场动画（关卡NPC提出谜题）
                                              ↓
                                      弹对话框 → 直接展示What三张知识卡
```

当前流程中，过场动画结束后直接跳到翻卡，缺少学者（玩家角色）与老学者（导师）之间的互动环节来引入知识。且"翻卡"的交互形式与游戏整体的古朴知识氛围不够贴合。

## 2. 修改后流程

```
点击节点 → 学者走到节点旁
              ↓
       [首次] 播放过场动画（关卡NPC提出谜题）
              ↓
       引导对话①：学者↔老学者（5句）
       学者提问 → 老学者引导
              ↓
       老学者引出三张卷轴
       卷轴在对话框上方水平逐一展开
       对话框同步显示老学者对当前卷轴的讲解
              ↓
       对话②：简短收尾（2句）
       学者感悟 → 老学者引导选择
              ↓
       选择关键卷轴（三选一确认）
```

两层布局：

```
┌────────────────────────────────────────────────┐
│                                                │
│         卷 轴 区（对话框上方独立区域）              │
│                                                │
│  ┌────── 水平展开 →──────┐                     │
│  │  📜 卷轴 ①            │                     │
│  │  "认知革命的本质是…"   │                     │
│  └────────────────────────┘                     │
│                                                │
├────────────────────────────────────────────────┤
│                        [ 跳过引导 ]              │
│  ┌──────┐           ┌──────┐                   │
│  │ 学徒 │           │老学者 │                   │
│  │ (亮)  │           │ (暗)  │                   │
│  └──────┘           └──────┘                   │
├────────────────────────────────────────────────┤
│  🧙 老学者                                      │
│  第一张卷轴告诉我们：智人的语言有一种              │
│  动物不具备的力量——                              │
│                                   ▾ 点击继续     │
└────────────────────────────────────────────────┘
         对话框区（始终固定在底部）
```

关键设计原则：
- **对话框不动**：对话阶段和卷轴阅读阶段，对话框始终固定在底部同一位置
- **卷轴在对话框上方展开**：不遮挡对话框，对话区作为卷轴的"讲解旁白"
- **卷轴阅读时不离开对话界面**：所有交互在同一个画面内完成
- **画像区+对话区合成一个整体对话框**：顶部两侧并排显示学徒和老学者头像（亮/暗区分当前说话者），下方宽条显示说话者名字和文本

## 3. 数据模型变更

### 3.1 WhatScroll 类型（替代 WhatCard）

```typescript
// 卷轴数据
export interface WhatScroll {
  type: 'definition' | 'example' | 'bridge';
  title: string;         // 卷轴标题（如"定义卷轴""例子卷轴""连接卷轴"）
  content: string;       // 卷轴正文内容
  mentorVoice: string;   // 卷轴展开时，老学者在对话框中的讲解台词
}
```

### 3.2 MentorPrompts 扩展

```typescript
export interface MentorPrompts {
  whatIntro: string;                      // 已有：老学者简短开场白
  whatDialogue?: DialogueLine[];          // 新增：对话阶段① 引导对话
  whatScrolls?: WhatScroll[];             // 新增：三张卷轴数据（替代 whatCards）
  whatWrapUp?: DialogueLine[];            // 新增：对话阶段② 收尾对话（阅读完卷轴后）
  how: string;
  why: string;
  system: string;
  finalReturn: string;
}

export interface DialogueLine {
  speaker: 'scholar' | 'mentor';
  text: string;
}
```

**向后兼容：**
- 如果节点没有 `whatDialogue`/`whatScrolls`，跳过对话+卷轴阶段，直接进入旧版 WhatCards
- `whatWrapUp` 可选，不提供则阅读完卷轴后直接进入"选择关键卷轴"

### 3.3 DialogPhase 扩展

```typescript
export type DialogPhase = 
  | "closed"
  | "intro_dialogue"       // 对话阶段①：学者↔老学者
  | "scroll_reading"       // 新增：卷轴阅读阶段（对话框在下，卷轴在上）
  | "intro_wrapup"         // 新增：对话阶段②：阅读完卷轴后的收尾对话
  | "loading"
  | "reading"
  | "what_confirm"
  | "question"
  | "feedback"
  | "final";
```

## 4. 详细界面设计

### 4.1 对话阶段①（intro_dialogue）

纯对话框交互，按现有 DialogBox 布局：

```
┌────────────────────────────────────────────────┐
│                        [ 跳过引导 ]              │
│  ┌──────┐           ┌──────┐                   │
│  │ 学徒 │           │老学者 │                   │
│  │ (亮)  │           │ (暗)  │                   │
│  └──────┘           └──────┘                   │
├────────────────────────────────────────────────┤
│  🧙 老学者                                      │
│  区别不在有没有语言，而在语言的种类。              │
│  动物的语言只能说真实的东西……                     │
│                                   ▾ 点击继续     │
└────────────────────────────────────────────────┘
```

- 右上角 **「跳过引导」** 链接（小字，与头像栏同行）
- 上方头像区：学徒在左，老学者在右，当前说话者头像高亮（亮），对方变暗 40%（暗）
- 下方对话区：第一行显示说话者名字 + emoji（如 🧙 老学者 / 🧑 学徒），逐字打字（45ms/字）
- 未完成时点击→立即显示全文；已完成后点击→跳到下一句
- 头像切换：当说话者切换时，上方头像的亮/暗状态对调
- 老学者最后一句说完后→**自动触发卷轴阶段**

#### 卷轴交互规则

| 规则 | 说明 |
|------|------|
| 展开动画 | 卷轴从中心向左右两侧水平展开（类似画卷展开），持续 600ms |
| 对话框头像区 | 老学者讲解时，老学者头像高亮（亮），学徒变暗（暗） |
| 对话框内容区 | 卷轴展开的同时，对话框显示 `scroll.mentorVoice` 逐字打字，开头显示 🧙 老学者 |
| 点击推进 | 对话框内点击：打字未完成→显示全文；已完成→下一张卷轴 |
| 末张完成 | 最后一张卷轴点击后→自动切换到对话阶段② |
| 跳过 | 右上角"跳过引导"仍然可用，跳过全部阶段直接到选择关键卷轴 |
| 进度指示 | 卷轴底部显示 `● ● ○` 进度点 + `1/3` 数字 |

#### 卷轴视觉规范

| 元素 | 设计 |
|------|------|
| 卷轴背景 | 米白/淡黄羊皮纸纹理，上下带有深色木轴装饰（像素风 4px 圆条） |
| 卷轴标题 | 顶部缎带标签，颜色按类型区分（定义=金棕、例子=草绿、连接=紫色） |
| 正文 | 16px 像素字体，深褐色 #492310，留白舒适 |
| 展开动画 | CSS transform: scaleX(0) → scaleX(1) 从中心向两侧，配合 translate3d |
| 收卷动画 | 切换到下一张时当前卷轴向中心收卷（scaleX(1) → 0），新卷轴展开 |
| 木轴细节 | 上下边缘 2px 深色描边 + 两端各一个 6×6 的轴头装饰点 |

  最后一句（老学者）结束后出现按钮：

┌────────────────────────────────────────────────┐
│                        [ 跳过引导 ]              │
│  ┌──────┐           ┌──────┐                   │
│  │ 学徒 │           │老学者 │                   │

- 共 2 句：学者感悟 → 老学者引导（头像在学者和老学者之间切换一次）
- 最后一句（老学者）话音结束后，对话区底部显示 **「选一张最关键的卷轴 →」** 按钮
- 点击后进入 `what_confirm` 三选一阶段（复用现有 WhatCards 的选择交互，但卡片换为卷轴缩略图）

#### ① 认知革命 · 篝火洞穴

**谜题：** 是什么让智人和其他动物有了根本的不同？

**引导对话：**

| # | 说话者 | 台词 |
|---|--------|------|
| 1 | 学者 | "老爷爷问'智人到底哪里不一样'……我第一反应是'更聪明'，但这个答案好像太敷衍了。" |
| 2 | 老学者 | "你感觉没错。智人的大脑不比尼安德特人大，真正的差别不是硬件，而是我们用的'软件'——语言。" |
| 3 | 学者 | "动物不也有语言？蜜蜂跳舞、猴子报警，这不也是在交流吗？" |
| 4 | 老学者 | "区别在种类。动物的语言只能说真实的东西——'那边有吃的'。而智人的语言能说'不存在'的东西。" |
| 5 | 老学者 | "来看看这三张卷轴吧——它们会告诉你，为什么能'讲虚构故事'，才是智人真正的武器。" |

**三张卷轴：**

| 类型 | 标题 | 正文 | 老学者讲解台词 |
|------|------|------|---------------|
| 定义 | 认知革命的定义 | 认知革命的本质不是大脑突变，而是一场语言使用方式的革命。智人获得了"虚构"的能力——能用语言描述不存在的事物，从此成为"讲故事的物种"。 | "第一张卷轴告诉我们：认知革命的关键不是智人变聪明了，而是语言变了——智人开始能谈论没见过、没摸过、不存在的东西。" |
| 例子 | 篝火边的陌生人 | 一群互不相识的原始人围坐在篝火旁，他们语言不通、部落不同，但都相信同一位"森林之灵"的神话。因为这个共同相信的故事，他们放下了武器，并肩协作。 | "第二张卷轴画出了那个场景：一群陌生人因为相信同一个神话而放下武器。这就是虚构故事的力量——让不认识的也能合作。" |
| 连接 | 从传说到公司 | 今天的跨国公司、国家、法律体系——其运作原理和远古人类围坐篝火讲神话，本质上完全一样：一群人共同相信一个虚构的实体。 | "最后这张最有意思：今天的大公司、国家，和远古神话的结构是一样的——都是大家共同相信一个看不见的东西。看完你有没有觉得，现代世界没那么'现代'？" |

**收尾对话：**

| # | 说话者 | 台词 |
|---|--------|------|
| 1 | 学者 | "原来如此……语言不只是交流工具，还能创造'共同相信的东西'。那今天的社会，其实也是建立在这种能力之上。" |
| 2 | 老学者 | "你已经开始看到连接了。现在选一张你觉得最关键的卷轴——它会成为我们继续深入的起点。" |

#### ② 农业革命 · 金色麦田

**谜题：** 为什么定居务农的我们，反而比流浪的猎人还要累？

**引导对话：**

| # | 说话者 | 台词 |
|---|--------|------|
| 1 | 学者 | "农夫说日子比流浪时还苦……农业不是人类最伟大的进步之一吗？" |
| 2 | 老学者 | "对'人类'这个物种来说是进步——人口翻了百倍。但对'每一个具体的人'呢？每天从早到晚种地，吃的东西就那几样，还动不动饿死。" |
| 3 | 学者 | "所以说，对个体来说是更辛苦了……但为什么大家还是要种地？" |
| 4 | 老学者 | "因为开弓没有回头箭。一旦开始种地、人口变多、变成村落，你就回不去采集生活了。农业就像一桩糟糕的交易。" |
| 5 | 老学者 | "很多历史学家管这叫'史上最大的骗局'。到底是人驯化了小麦，还是小麦驯化了人，来看看这三张卷轴。" |

**三张卷轴：**

| 类型 | 标题 | 正文 | 老学者讲解台词 |
|------|------|------|---------------|
| 定义 | 农业革命的定义 | 农业革命是智人从采集狩猎转向定居种植的历史转变。但这并非一次聪明的选择，而是一条没有退路的陷阱——人口增长后，再也回不去旧生活了。 | "第一张卷轴的定义很直接：农业革命不是'进步'，而是一条不归路。人口增长之后，回头路就断了。" |
| 例子 | 小麦的胜利 | 那个满头大汗的农夫，一天的工作量远超采集者，但吃得还不如从前。他的骨骼更差、牙齿更坏、传染病更多。小麦让他困在田里，走不掉了。 | "看这张卷轴上的农夫——他比采集者累得多，但身体反而更差。小麦赢了他。到底谁驯化了谁？" |
| 连接 | 你也在种地？ | 今天的"上班"和农业革命本质上一样：为了稳定的食物来源（工资），牺牲了自由和多样性。朝九晚五的现代人，和弯腰种地的农夫，没有本质区别。 | "最后这张卷轴做了一个现代连接：你有没有觉得每天上班和种地挺像的？为了稳定收入放弃自由——这个模式从农业革命就开始了。" |

**收尾对话：**

| # | 说话者 | 台词 |
|---|--------|------|
| 1 | 学者 | "原来农业不一定是好事……那我每天上班、加班，是不是也是一种'现代版农业陷阱'？" |
| 2 | 老学者 | "好问题。带着这个角度选一张最关键的卷轴——它会带我们看得更深。" |

#### ③ 货币 · 热闹市集

**谜题：** 为什么一张不能果腹的纸，能买到最真实的食物？

**引导对话：**

| # | 说话者 | 台词 |
|---|--------|------|
| 1 | 学者 | "一张纸凭什么能买到包子？明明它自己就是一张纸。这个问题我一直觉得有点神奇。" |
| 2 | 老学者 | "神奇就对了。金钱的秘密就在于：**所有人都相信它值钱**。不是因为它本身有什么价值，而是因为每个人都这么信。" |
| 3 | 学者 | "那不就是'大家约定好'？和法律、规则差不多吗？" |
| 4 | 老学者 | "对，但比法律更厉害。法律只能管一个国家的范围。而金钱——你拿一张人民币，在美国、非洲、南极都能花。世界上没有第二种东西有这么大的信任网络。" |
| 5 | 老学者 | "金钱是人类最成功的'共同想象'，比任何宗教的信徒都多。来看看这三张卷轴。" |

**三张卷轴：**

| 类型 | 标题 | 正文 | 老学者讲解台词 |
|------|------|------|---------------|
| 定义 | 货币的定义 | 货币的本质不是黄金也不是纸张，而是一个信任系统。一切货币，不论贝壳、金币还是数字代码，都是"共同想象的信用"——所有人相信它有价值，它就有价值。 | "第一张卷轴点明了核心：金钱不是东西，是信任。贝壳、金币、纸币——形式不重要，重要的是所有人都信。" |
