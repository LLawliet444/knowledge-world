# Knowledge World Layer Loops

四首 BGM 已重做为“无低频压力版本”。最低音尽量不低于 C3，移除 cello drone、低频 pulse、low-breath、低八度 pad，并额外用 170Hz high-pass 作为安全网。前端优先使用 `ogg`。

## 文件

- `knowledge_world_main_loop.wav` / `.ogg` / `.mp3`：What / 主地图，48 秒，80 BPM，44.1kHz stereo
- `knowledge_world_how_loop.wav` / `.ogg` / `.mp3`：How，48 秒，80 BPM，44.1kHz stereo
- `knowledge_world_why_loop.wav` / `.ogg` / `.mp3`：Why，48 秒，80 BPM，44.1kHz stereo
- `knowledge_world_system_loop.wav` / `.ogg` / `.mp3`：System，48 秒，80 BPM，44.1kHz stereo

## 设计

- What：轻风感、harp、celesta，保持开放和好奇。
- How：重复钢琴动机和轻木质 click，表现结构形成。
- Why：慢速钢琴和轻 bell，不使用 drone 或低频脉冲。
- System：soft pad、glass bell、少量高频星光，不使用 choir/drone。
- 建议前端切层时使用 1.5 秒 crossfade，避免硬切。
