# Knowledge World Layer Loops

四首 BGM 已重做为 `No Pad / No Drone / No Reverb` 版本。所有持续铺底声、delay、reverb、低频 pulse、low-breath、cello drone、低八度 pad 均已移除；只保留短衰减的 piano、celesta、harp、wood click 和留白，并统一使用 300Hz high-pass 优先去除嗡嗡声。前端优先使用 `ogg`。

## 文件

- `knowledge_world_main_loop.wav` / `.ogg` / `.mp3`：What / 主地图，48 秒，80 BPM，44.1kHz stereo
- `knowledge_world_how_loop.wav` / `.ogg` / `.mp3`：How，48 秒，80 BPM，44.1kHz stereo
- `knowledge_world_why_loop.wav` / `.ogg` / `.mp3`：Why，48 秒，80 BPM，44.1kHz stereo
- `knowledge_world_system_loop.wav` / `.ogg` / `.mp3`：System，48 秒，80 BPM，44.1kHz stereo

## 设计

- What：harp 与 celesta 的点状音符，保持开放和好奇。
- How：短钢琴动机和轻木质 click，表现结构形成。
- Why：大量留白、慢速短钢琴，不使用任何持续背景层。
- System：短 bell 与高音 sparkle，不使用 glass pad / choir / drone。
- 这一版优先解决嗡嗡声和听觉压迫，氛围会比之前更轻、更干净。
